# Git Flow Visualizer / Git Flow 可视化

## English

### Overview

Git Flow Visualizer is an interactive walkthrough for practicing the Git Flow branching model.
It includes a language toggle (English/中文), a guided step list, and a visual commit graph with terminal-style logs.
Use the Language toggle in the left sidebar header to switch between English and 中文.

### Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

### Deploy to GitHub Pages

This repository includes a GitHub Actions workflow that builds and deploys the site on pushes to `main`.
Enable GitHub Pages in the repository settings and select **GitHub Actions** as the source.

## 中文

### 简介

Git Flow 可视化工具用于交互式学习 Git Flow 分支模型。
它提供语言切换（English/中文）、分步教程、以及带终端日志的可视化提交图。
可以使用左侧栏顶部的语言切换按钮在 English 和 中文之间切换。

### 本地运行

**前置条件：** Node.js

1. 安装依赖：
   `npm install`
2. 启动应用：
   `npm run dev`

### 部署到 GitHub Pages

仓库自带 GitHub Actions 工作流，会在推送到 `main` 时构建并部署站点。
在仓库设置中启用 GitHub Pages，并将来源选择为 **GitHub Actions**。
